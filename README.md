# AgentSkills

A collection of reusable skills, utilities, and examples for building autonomous agents.

## Contents

- **skills/** - Implementations of common agent capabilities (planning, reflection, tool use)
- **examples/** - Minimal working examples showing how to combine skills
- **utils/** - Helper functions for logging, state management, etc.

## Getting Started

Clone this repository and explore the skills directory to find building blocks for your own agent.

## Contributing

Feel free to open issues or submit pull requests with new skills or improvements.
